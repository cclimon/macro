================================================================================
  MACRO / RATES TRADING SYSTEM  —  Repository Summary
  GitHub: https://github.com/cclimon/macro.git
  Last updated: 2026-06-26
================================================================================

  Multi-asset quantitative trading system covering STIR rates, G10 FX macro,
  FX volatility, and realized vs implied vol analysis. Python signal engines
  feed both a Streamlit dashboard and a React/Vite web frontend.

  MODULES AT A GLANCE
  -------------------
  stir-engine/         SOFR/FF futures algo: mean reversion (OU), momentum,
                       OSINT signals, Markov-switching regime, backtest/walk-fwd

  macro/               G10 FX signals via Bloomberg: technical (RSI/MACD/BBands),
                       carry (fwd-implied vs RV), macro (PMI/CPI/rate diffs).
                       Streamlit dashboard in macro/dashboard/app.py

  dashboards/          React/Vite SPA: Israel Gov cabinet, TIC flows,
                       US Fed liquidity (reserves, RRP, TGA, SOFR-IORB, MOVE)

  rv-analysis/         FX options RV vs IV: Yang-Zhang & EWMA estimators,
                       IV-RV spread percentile signals, USDINR RSI backtest

  fx-vol-monitor/      Intraday FX vol + carry heatmap monitor, PNG + CSV export
  fx-vol-bands/        Vol regime-based entry/exit bands per currency pair
  MF-Engine/           Multi-factor STIR engine (extension of stir-engine)
  correlation-machine/ Correlation framework (stub/in progress)

  ROOT FILES
  ----------
  liquidity_data.py    Fetches Fed liquidity data (Bloomberg/FRED) -> dashboards/public/data/
  macro_charts.py      GDP per worker charts via FRED + matplotlib -> output/macro/
  trade_ideas.json     Manual trade ideas log
  MODELS-RUN-HOWTO.txt How to run each model

  DATA FLOW
  ---------
  Bloomberg / FRED / yfinance / CME
    -> Python engines (stir-engine, macro, rv-analysis)
    -> JSON / Parquet cache
    -> Streamlit dashboard  OR  React/Vite SPA (dashboards/)
